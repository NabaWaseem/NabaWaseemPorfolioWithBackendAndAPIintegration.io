# ✨ Features

- 📱 **Responsive Design**

  - UI/UX Design

# 🔌 API Integration (Mock Mode)

This project includes a **mock backend** to simulate API responses.

### How it works

- A flag inside `script.js` controls whether mock data is used.
- By default, it is set to **true**.
- Portfolio sections such as:
  - Profile
  - Skills
  - Projects
  - Education
  - Testimonials

are loaded from a local JavaScript object.

When connecting to a real backend:

1. Change the flag to `false`.
2. Replace the mock data with real API endpoints.

This setup makes the transition to a real backend smooth and straightforward.
